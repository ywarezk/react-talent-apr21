# Loops

## For

```
for (let i = 0; i<10; i++) {
	
}
```

```
for (let i of someArray) {
	
}
```

```
// usually not used
for (let i in someObject) {
	
}
```

## forEach

```
[1,2,3].forEach((singleNum) => {
	
});
```

## map

```
[1,2,3].map((num) => `hello world ${num}`)
// [hello world 1, hello world 2, hello world 3]
```

## while, do while

```
while(condition) {
	do this
}
```

```
do {
	
} while(condition)
```
